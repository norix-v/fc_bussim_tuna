tuna_can

------------
What's this?
------------

A simple debugger for FC cartridges for tuna (or kazzo).
It is a tool to write, read, and dump data to an appropriate address.
Please use it as a companion when creating anago scripts.

You can see how to use it because the help is displayed at startup.
As a bonus, it has a history function above and below the cursor keys 
(up to 64 entries).

------------
How to build
------------
You can compile with Visual Studio Community 2019.
